1：平衡符号
leetcode上面OJ：

https://leetcode-cn.com/problems/valid-parentheses/submissions/

思路：

既然要判断左右括号的匹配，那么每一个花括号，中括号，小括号等等，只要出现了左括号，那么就一定要匹配上其右括号。

那么这样，想到了什么？“先进后出”貌似是最好的方法
定义一个栈，如果是 ’ ( ‘ ‘ [ ‘ ‘ { ‘ 等左边括号，那么就让它们统统入栈
